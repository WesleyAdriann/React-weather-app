# Weather App

Aplicação de clima utilizando open wather.

## Utilização
### Método 1
[Node](https://nodejs.org/)
```
npm start
```
### Método 2
[Docker](https://www.docker.com)
```
docker-compose up --build
```

Para informações de:

* [React](https://reactjs.org)
* [Open Weather](https://openweathermap.org/api)
